

# The Chess Game

### How to run the game?

If Flask 2.0 and Python 3.8 is installed then navigate to the projects folder and type the following into the terminal.

Linux, macOS:
```shell
$ export FLASK_APP=server
$ flask run --host=0.0.0.0 --port=8000
```
Windows:
```shell
> set FLASK_APP=server
> flask run --host=0.0.0.0 --port=8000
```
